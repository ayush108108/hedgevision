## [0.1.0] - 2026-03-31

### Added
- Initial public open-source release
- Full-stack statistical arbitrage platform (FastAPI backend + React frontend)
- Local-first architecture with SQLite (no external services required)
- Cointegration analysis, correlation tracking, and pairs discovery
- Paper trading with realistic order execution simulation
- Real-time dashboard with live analytics
- Flexible LLM integration (Ollama, OpenAI, Anthropic)
- CCXT broker abstraction for live exchange connectivity
- CLI and MCP Tools for automation
- Docker support for containerized deployment
- Comprehensive test suite (unit + integration)
- Full documentation (SETUP, ARCHITECTURE, DEPLOYMENT, SCRIPTS)
- CONTRIBUTING.md, CODE_OF_CONDUCT.md, SECURITY.md
